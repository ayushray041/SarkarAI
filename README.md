## 📖 Overview

Government schemes and official documents contain valuable information, but understanding them often requires significant time and effort. Complex language, lengthy documentation, and scattered eligibility criteria create barriers for citizens seeking benefits and opportunities.

**SarkarAI** is an AI-powered Government Document Intelligence Platform that bridges this gap by transforming complex government documents into simple, personalized, and actionable insights.

Users can upload government documents and instantly receive AI-generated summaries, eligibility information, important deadlines, required documents, and answers to their questions through an intelligent conversational interface.

---

## 🚨 Problem Statement

Millions of citizens struggle to access and understand government schemes because:

- Government documents are lengthy and difficult to interpret.
- Eligibility requirements are often unclear.
- Important dates and deadlines are easy to miss.
- Citizens must manually search through multiple sources for information.
- Technical and legal terminology creates accessibility barriers.
- Finding scheme-specific answers requires significant effort.

These challenges result in low awareness, missed opportunities, and inefficient access to public services.

---

## 💡 Our Solution

SarkarAI simplifies the entire process by leveraging Generative AI and Natural Language Processing.

Instead of reading hundreds of pages, users can:

1. Upload a government document.
2. Receive an AI-generated summary.
3. View extracted eligibility criteria.
4. Identify required documents.
5. Track important dates and deadlines.
6. Interact with an AI assistant to ask questions in natural language.

This transforms complex documentation into a user-friendly experience accessible to everyone.

# ✨ Core Features

## 📄 Intelligent Document Analysis

Upload government schemes, notifications, policies, and official documents to receive:

- AI-generated summaries
- Key highlights

- Simplified explanations

## 🤖 AI-Powered Conversational Assistant

Users can ask questions such as:

- "Who can apply for this scheme?"
- "What documents are required?"
- "When is the application deadline?"
- "Am I eligible?"

The AI provides contextual responses based on the uploaded document.

---

## ✅ Eligibility Assessment

Automatically extracts and analyzes:

- Age requirements
- Income criteria
- Educational qualifications
- Category-specific conditions
- Other eligibility rules

Helping users quickly determine whether they qualify.

---

## 📅 Important Dates Extraction

Identifies and highlights:

- Registration deadlines
- Application start dates
- Verification schedules
- Result announcements

Ensuring users never miss critical timelines.

---

## 📋 Required Documents Identification

Automatically extracts required documents such as:

- Aadhaar Card

- Residence Proof

And presents them in a structured format.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/SarkarAI.git
```

## Navigate to Project

```bash
cd SarkarAI
```

## Install Dependencies

```bash
npm install
```

## Configure Environment Variables

Create a `.env.local` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

## Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
